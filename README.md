# AmazonClone
This project is a frontend UI clone of Amazon, built using HTML & CSS. It replicates the look and feel of the Amazon website but does not include backend functionality.

Features
Home Page: A responsive homepage with banners, product categories, and deals.
Product Page: A well-structured product details page with images, descriptions, and pricing.
Cart Page: A shopping cart interface for users to view selected items.
Responsive Design: Works on desktops, tablets, and mobile devices.
Technologies Used
HTML for structure
CSS for styling and layout
